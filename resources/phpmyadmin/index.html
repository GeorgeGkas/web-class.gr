Εγκατάσταση PhpMyAdmin

Πέτρος Αγγελάτος, για το Σεμινάριο: Ανάπτυξη Web Εφαρμογών 2010

Οι παρακάτω οδηγίες έχουν δοκιμαστεί σε Windows 7. Για την εγκατάσταση MySQL, PHP και Apache υπάρχει ξεχωριστός οδηγός.

Αν έχεις PHP έκδοση παλιότερη της 5.3 θα χρειαστεί να την αναβαθμίσεις.

  1. Κατέβασε το αρχείο εγκατάστασης

    Μπες στη σελίδα http://www.phpmyadmin.net/home_page/downloads.php και κατέβασε την πιο πρόσφατη αγγλική έκδοση του phpMyAdmin.

    Εγώ κατέβασα αυτό το αρχείο.

  2. Εγκατάσταση

    Κάνε extract τα περιεχόμενα του αρχείπου κατέβασες στον φάκελο htdocs της εγκατάστασης του Apache:

    Εγώ έκανα extract τον φάκελο με όνομα phpMyAdmin-3.3.8.1-english και τον μετονόμασα σε pma

  3. Ρύθμιση

    Το phpMyAdmin θα πρέπει να δουλέψει χωρίς κάποια επιπλέον ρύθμιση

    1. Άνοιξε τον φυλλομετρητή σου
    2. Πήγαινε στη σελίδα http://localhost/pma/
    3. Θα πρέπει να εμφανιστεί το login screen του phpMyAdmin. Συμπλήρωσε root στο username και στο password το password που όρισες όταν εγκαθιστούσες την MySQL
    4. Συνδέσου στην βάση δεδομένων

  4. Δημιουργία βάσης δεδομένων

    Μία web εφαρμογή καλό είναι να συνδέεται στην MySQL όχι ως root αλλά ως κάποιος άλλος χρήστης που έχει δικαίωμα πρόσβασης μόνο στη βάση που αφορά την εφαρμογή:

    Ακολούθησε τα παρακάτω βήματα όταν έχεις αποφασίσει για το όνομα και το σχήμα της βάσης δεδομένων σου

    1. Συνδέσου με το phpMyAdmin χρησιμοποιοώντας τον χρήστη root
    2. Στο πλαίσιο κάτω από το Create new database συμπλήρωσε το όνομα που θες να έχει η βάση δεδομένων
    3. Στο Collation δεξιά από το όνομα επέλεξε utf8_unicode_ci
    4. Πάτα το κουμπί Create
    5. Πάτα το κουμπί Home από την αριστερή στήλη
    6. Πήγαινε στην καρτέλα Priviledges
    7. Κάνε κλικ στο κουμπι Add a new User
    8. Στο πεδίο username συμπλήρωσε το username της επιλογής σου.
    9. Στο πεδίο host επέλεξε Local
    10. Συμπλήρωσε το password που θες να έχει ο χρήστης που δημιουργείς
    11. Πάτα το κουμπί Go
    12. Στο πλαίσιο Database-specific privileges στο πεδίο Add privileges on the following database επέλεξε την βάση δεδομένων που δημιούργησες
    13. Θα μεταφερθείς στις ρυθμίσεις για τα δικαιώματα του χρήστη πάνω στη συγκεκριμένη βάση δεδομένων
    14. Στο πλαίσιο Database-specific privileges κάνε κλικ στο Check all
    15. Πάτα το κουμπί Go
    16. Κάνε κλικ στο πράσινο εικονίδιο που γράφει Exit στην αριστερή στήλη για να κάνεις logout
    17. Κάνε login με το όνομα χρήστη και password που δημιούργησες

Πλέον είσαι σε στάδιο να δημιουργήσεις τους πίνακες της βάσης σου.

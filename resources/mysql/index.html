Εγκατάσταση MySQL

Αλέξης Μπρέζας, για το Σεμινάριο: Ανάπτυξη Web Εφαρμογών 2010

Σε Windows

Οι παρακάτω οδηγίες έχουν δοκιμαστεί σε Windows 7. Για την εγκατάσταση PHP και Apache υπάρχει ξεχωριστός οδηγός.

Αν έχεις PHP έκδοση παλιότερη της 5.3 θα χρειαστεί να την αναβαθμίσεις.

  1. Κατέβασε το αρχείο εγκατάστασης

    Μπες στη σελίδα http://dev.mysql.com/downloads/mysql/ και πάτα Download sτο Windows MSI Installer Essentials. Θα μεταφερθείς σε φόρμα εισόδου ή εγγραφής στο mysql.com, αλλά μπορείς να το ξεπεράσεις πατώντας στο link "No thanks" από κάτω. Από εκεί πάτα στο HTTP download link του NTUA.

    Εγώ κατέβασα αυτό το αρχείο.

  2. Εγκατάσταση

    Με το αρχείο εγκατάστασης η διαδικασία είναι η συνηθισένη:

    1. Πάτησε Next
    2. Επίλεξε Typical
    3. Πάτησε Install

    Στο παράθυρο διαφήμισης του MySQL Enterprise προχώρησε παρακάτω και πάτησε Finish ώστε να ανοίξει το παράθυρο ρύθμισης (Configuration Wizard).

  3. Ρύθμιση

    Η ρύθμιση είναι καλό να γίνει αναλυτικά, κυρίως για να έχουμε UTF-8 βάσεις δεδομένων εκ προεπιλογής. Ακολούθησε:

    1. Πάτησε Next
    2. Επίλεξε Detailed Configuration και πάτησε Next.
    3. Επίλεξε Developer Machine και μετά Multifunctional Database.
    4. Επίλεξε το δίσκο που θες να χρησιμοποιεί η MySQL. Αν δεν είσαι σίγουρος απλά πήγαινε παρακατω.
    5. Επίλεξε Decision Support και πάτησε Next.
    6. Πρόσεξε ότι έχεις τα "Enable TCP/IP Networking" και "Enable Strict Mode" ενεργοποιημένα και Next.
    7. Εδώ αλλάζουμε την αρχική επιλογή σε "Best Support for Multilangualism".
    8. Ενεργοποίησε τα "Install as Windows Service" και "Include Bin Directory in Windows PATH".
    9. Φτιάξε ένα λογαριασμό διαχειριστή (root). To "Current Password" άφησέ το κενό. Προτείνεται να μην βάλεις πολύ εύκολο password.
    10. Τέλος, ζήτα από το πρόγραμμα να εκτελέσει τις διαδικασίες και όταν τελειώσει πάτησε finish.

  4. Σύνδεση με την PHP

    Ο δαίμονας της MySQL έχει πλέον εγκατασταθεί και ρυθμιστεί σωστά. Θα πρέπει να ενεργοποιήσεις τώρα τις συναρτήσεις της PHP που επιτρέπουν την σύνδεση με την MySQL:

    1. Τρέξε το phpinfo() όπως περιγράφουμε στον προηγούμενο οδηγό για να βρεις την θέση του php.ini
    2. Επεξεργάσου το αρχείο php.ini. Ενδέχεται να χρειάζεται να αλλάξεις τις ιδιότητες του αρχείου ώστε να ενεργοποιήσεις την δυνατότητα εγγραφής.
    3. Βρες την γραμμή που γράφει "extension=php_mysql.dll" και βγάλε το ";" που υπάρχει στην αρχή.
    4. Βεβαιώσου ότι η ρύθμιση "magic_quotes_gpc" είναι ρυθμισμένη σε "Off".
    5. Βεβαιώσου ότι η ρύθμιση "register_globals" είναι ρυθμισμένη σε "Off".
    6. Κάνε επανεκκίνηση του Apache.

Πλέον είσαι σε στάδιο να συνδεθείς στην MySQL μέσα από την PHP με τις συναρτήσεις που έχουμε δείξει.

Σε Ubuntu

Τρέξε την ακόλουθη εντολή και ακολούθησε τις οδηγίες:

sudo apt-get install mysql php5-mysql
